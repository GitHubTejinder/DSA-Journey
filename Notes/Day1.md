What is an Algorithm?
- Algorithm is set of instructions for accompolishing a task.

What is Source Code?
- Source code is what we write in a computer for any programming language.

What is Compilation?
- Computer only understands the Machine code (which is Binary data) so Source code that we write needs to be converted to Machine code and process of converting the Source code to Machine code is called Compilation.

What is an Efficient Algorithm?
- Algorithm that can solve the problem or accompolish the task in minimum numbers of iterations or quickly is the Efficient algorithm.

What is Big-O?
- Is the notation to define the time that the algorithm takes to solve the problem or accompolish the task.

Why is Binary Search Faster?
- Because it eliminates the half the options with each iteration.
